# 🎨 CSS 样式

本目录用于存放公共 CSS 样式文件。

## 📋 规划中的文件

| 文件 | 说明 |
|------|------|
| `variables.css` | CSS 变量定义（颜色、字体等） |
| `common.css` | 公共样式（布局、按钮等） |
| `graph.css` | 图形相关样式 |
| `theme-dark.css` | 深色主题 |
| `theme-light.css` | 浅色主题 |

## 🎯 设计目标

从 HTML 模板中抽取公共样式，实现：

- 统一的设计语言
- 主题切换支持
- 更好的可维护性
